### 星火讯飞大模型 SDK

## 简介

本 SDK 基于星火讯飞大模型，提供了一套便捷易用的接口，帮助开发者轻松调用星火讯飞提供的各种 AI 能力。该 SDK 经过精心设计，易于使用，只需修改 config.yml 文件中的配置信息，即可调用运行。

## 快速上手

安装 SDK
修改 config.yml 文件中的配置信息
参照调用示例，即可开始使用
